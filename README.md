# Tkinter & SQLite Authentifizierungssystem

Eine saubere und sichere Python-Desktop-Anwendung, die ein Benutzer-Login-System mit einer grafischen Benutzeroberfläche (Tkinter) und einer SQLite-Datenbank bietet.

## Funktionen

* *Grafische Benutzeroberfläche:* Entwickelt mit Pythons nativer Tkinter-Bibliothek.
* *Datenbank-Persistenz:* Verwendung von SQLite3 zur sicheren Speicherung von Benutzerdaten.
* *Sichere Software-Architektur:* Klare Trennung der Datenbanklogik (utilisateurs.py) von der UI-Schicht (main.py).
* *Eingabevalidierung:* Überprüfung von leeren Feldern und saubere Authentifizierungsprüfung.

## Verwendete Technologien

* *Programmiersprache:* Python 3
* *GUI-Framework:* Tkinter
* *Datenbank:* SQLite3
* *Versionsverwaltung:* Git & GitHub

```bash
